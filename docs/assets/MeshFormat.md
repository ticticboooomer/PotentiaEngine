# Mesh Bundle Format

## Pseudo Format

#### Vertex:

`px py pz rgba tx ty nx ny nz gx gy gz`

###### Position

`f32 px`
`f32 py`
`f32 pz`

###### Color

`i32 rgba`

###### Texture Coordinate

`f32 tx`
`f32 ty`

###### Normals

`f32 nx`
`f32 ny`
`f32 nz`

###### Tangents

`f32 gx`
`f32 gy`
`f32 gz`


#### Indices

`i32 i`
